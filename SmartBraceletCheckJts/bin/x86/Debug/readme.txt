万魔声学PCBA自动测试系统V1.0.0
发布日期：2021.3.25 
发布人：Hanson 
发布内容：
1、支持MIC板：灵敏度-谐波失真测试、电气参数测试；
2、支持功放AMP板：电气参数测试（多通道电流、电压）、操作员工人判断测试；
3、支持WIFI板：WIFI 2.4G、5G的TX测试（矢量调制误差 EVM、平均功率 Pow、频率误差 FreqErr、频偏 模板 MaskErr）；
4、支持WIFI板：WIFI 2.4G、5G的RX测试（接收灵敏度、误包率）
5、支持WIFI板：BT蓝牙 TX测试；
7、支持生成测试日志文件；
8、支持测试项，标准范围等参数可配；
9、支持自动检测设备上电；
10、支持工序选择；
11、支持统计数，测试计时。
